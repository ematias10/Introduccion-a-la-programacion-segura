# Introducción a la Programación Segura - Repositorio de Clases

Este repositorio contiene el material práctico, laboratorios y ejercicios desarrollados para el curso de **Introducción a la Programación Segura (2026-1)**. Aquí se suben soluciones y explicaciones enfocadas en aplicar conceptos de programación con un enfoque preventivo y de seguridad.

## 🚀 Propósito del Repositorio

El objetivo es proporcionar ejemplos claros y prácticos de cómo implementar lógica de programación robusta para mitigar vulnerabilidades comunes y gestionar datos de forma segura desde las bases del lenguaje Python.

## 📂 Estructura del Proyecto

El contenido está organizado por unidades temáticas:

### [1. Fundamentos](1.%20Fundamentos/)
Introducción a la sintaxis de Python con aplicaciones de seguridad.
- **Conceptos**: Tipos de datos, variables y entrada/salida de datos segura.
- **Laboratorio**: Registro de activos de red y fragmentación de payloads.
- **Práctica**: Estimación de tiempos en ataques de fuerza bruta.

### [2. Comparaciones y Lógica](2.%20Comparaciones%20y%20Logica/)
Implementación de flujos de control y validación de seguridad.
- **Laboratorio**: Validación de credenciales (PIN) y enrutamiento lógico basado en tráfico de red.
- **Práctica**: Sistemas de autenticación VPN y control de acceso.

### [3. Ciclo While](3.%20Ciclo%20While/)
*(En desarrollo)* Estructuras de repetición aplicadas a la automatización de procesos de seguridad.

## 🛠️ Requisitos

- **Python 3.x**: Los scripts están diseñados para ejecutarse en versiones modernas de Python.
- **Editor sugerido**: VS Code.

## 📖 Cómo utilizar este material

Cada ejercicio dentro de las carpetas `Laboratorio` y `Practica` incluye comentarios que explican el razonamiento detrás de la solución, priorizando no solo que el código funcione, sino que sea seguro y manejable.
